# 1.Class View란?

클래스 기반 뷰(Class-Based Views, CBV)는 웹 사이트를 만들 때 사용하는 도구 중 하나로, 이 도구를 사용하면 웹 페이지가 어떻게 작동할지를 "클래스"라는 것을 통해 정리할 수 있어요. 클래스는 마치 레고 블록처럼, 웹 페이지를 만드는 데 필요한 여러 기능들을 조립해서 쉽게 사용할 수 있게 해줍니다.

예를 들어, 인터넷에서 뭔가를 검색할 때 보이는 그 페이지나, 글을 쓰는 페이지, 글 목록이 나열된 페이지 등을 만들 때, 모두 다르게 보이지만 사실은 비슷한 작업을 하고 있어요. 클래스 기반 뷰를 사용하면 이런 비슷한 작업들을 쉽게 재사용할 수 있어서, 같은 일을 반복해서 하지 않아도 돼요.

예를 들어서, 책을 소개하는 웹 페이지가 있다고 치면, 책의 목록을 보여주는 페이지, 책의 상세 내용을 보여주는 페이지 등이 필요할 거예요. 클래스 기반 뷰를 사용하면, 이런 페이지들을 만드는 기본 틀을 미리 만들어두고, 각각의 책에 맞게 조금씩 바꿔서 사용할 수 있어요.

또, 웹 페이지를 만들 때 사용자가 로그인을 했는지, 어떤 권한이 있는지 등을 확인해야 할 때도 있어요. 클래스 기반 뷰를 사용하면, 이런 확인 작업도 미리 만들어진 도구를 가져다 쓰기만 하면 되니까 훨씬 쉬워져요.

마지막으로, 클래스 기반 뷰는 웹 페이지를 만드는 규칙들을 잘 정리해 두기 때문에, 다른 사람이 나중에 그 코드를 봐도 무슨 일이 일어나는지 이해하기 쉬워요. 이렇게 해서 웹 페이지를 만들고 관리하는 일이 훨씬 수월해집니다.
